SEP11 Freedom Project

Context

SEP11 Freedom Project is the end of year project for my Junior year. I designed an 6-letter Wordle app in regards to issues of entertainment and educational purposes. What impact my app does is that there should be more widespread acknowledgement of educational apps that aims to prioritize on building skills rather than just to provide entertainment. Especially because a lot of people are on their phones and gaming has essentially lowered productivity, lack of learning experience, and less on strengthening cognitive minds.

Content

In my SEP11 Project, the main goal of the app is to guess the right word. This app does exactly that but you also have to think critically. The app can improve learning through learning about new words, expanding onto your current dictionary, and making use of the decisions since you are given 6 chances to guess the secret word.

Reflection

Making this project made me learn about ReactJS and also put Javascript to the test. Although there was a lot of changes I could use to make my app stand out, I have to accept that it's better to make the app functional rather than trying to make it look appealing. Another thing I would have better made this project original is by adding more remixes rather than just making the word one letter longer than the original Wordle game.

One challenge I had was keeping track of multiple changes. Since React uses states and props, states are constantly changing and when my code expands, it's difficult to keep track of what each line of code does. A solution to this was basically going back and tracking the changes manually line by line.

Another challenge I had was trying to add new codes but forgetting some of the things I learned in my previous learning log. So having my learning log show the steps of each tinkering and notes on what each code does really helped me out with piecing new ideas, new codes that follow the things I learned when following React's documentation. Also, I didn't download React so I had to use Babel which made it difficult when I started using React for the first time.

Some takeaways I had after presenting in class and during the Expo is that I should feel more confident when speaking to judges because I get so nervous when looking at the audience. Also, the volume of my voice should be higher since in the Expo, there was a bunch of noises in the background that hinders my presentation.

One next step I have is to invest some time to add what I couldn't after the Expo.
